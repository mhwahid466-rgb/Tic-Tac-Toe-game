# 🎮 Tic Tac Toe Game

A simple and modern Tic Tac Toe game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 👥 Two player game (Player X vs Player O)
* 🏆 Winner announcement
* 🔄 Reset & New Game button
* 🎨 Modern UI design
* ⚡ Fast and responsive gameplay

## ▶️ How to Play

1. Open the game in your browser
2. Player X starts first
3. Take turns placing X and O
4. First to align 3 marks (row, column, diagonal) wins
